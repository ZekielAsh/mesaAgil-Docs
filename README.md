Se dedicara este repositorio a la presentación de información del proyecto. 

El grueso del codigo estara ubicado en otros repositorios.



## Objetivo
Crear una aplicación que permita mejorar la experiencia del comensal en un restaurante agilizando el proceso de encargo de comida. También se busca permitir al establecimiento obtener información respecto del desempeño de su servicio.

## Descripción
Proyecto de aplicación para restaurantes con un sistema de pedidos accesible, conexión entre diferentes secciones del establecimiento y recolección de estadísticas.

## Tecnologías
- Java 17
- React Native
- Spring Boot 4.0
